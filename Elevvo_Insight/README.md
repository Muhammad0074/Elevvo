# Elevvo Insight

> Smart Analytics for Growing Teams — a demo landing page and UI prototype.

## Overview

Elevvo Insight is a lightweight demo landing page showcasing a fictional analytics product focused on product-led teams. This repository contains a static HTML/CSS/JS prototype intended for design review, iteration, and small front-end experiments.

## Contents

- `index.html` — Landing page markup and content.
- `styles.css` — Styles for layout, components, and theming.
- `script.js` — Lightweight behaviors (theme toggle and form handling).

## Features Highlighted

- Hero with live product overview panel
- Feature grid and testimonial sections
- Simple pricing cards and CTA email capture form
- Theme toggle (light/dark) with small JS behavior

## Usage

1. Open `index.html` in a browser to view the demo.
2. For local development, serve the folder with any static server (optional):

```bash
# using Python 3
python -m http.server 8000

# then open http://localhost:8000
```

## Development

- Edit `index.html`, `styles.css`, and `script.js` to iterate on layout and behavior.
- Keep changes focused and minimal; this repo is a small static prototype.
