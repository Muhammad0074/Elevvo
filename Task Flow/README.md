# Task Flow

## Project Overview

Task Flow is a small static web project containing a single-page front-end implementation. It serves as a minimal demonstration of a client-side UI using HTML, CSS, and JavaScript.

## Features

- Simple, self-contained static site
- Clear separation of markup, styles, and behavior

## Files

- [index.html](index.html) — The main HTML entry point.
- [styles.css](styles.css) — Stylesheet for layout and presentation.
- [script.js](script.js) — Client-side JavaScript for interactive behavior.

## Prerequisites

- A modern web browser (Chrome, Edge, Firefox, Safari).
- Optional: Python or Node.js to run a local static server.

## Setup & Local Preview

To preview the site, open `index.html` directly in your browser, or serve the directory using a simple static server.

Using Python 3 (recommended):

```bash
python -m http.server 8000

# then open http://localhost:8000 in your browser
```

Using Node.js with http-server (if installed):

```bash
npx http-server -p 8000

# then open http://localhost:8000 in your browser
```

## Usage

1. Start a local server (optional). 2. Open [index.html](index.html) in the browser. 3. Interact with the UI; changes to `script.js` and `styles.css` will be reflected after reload.
